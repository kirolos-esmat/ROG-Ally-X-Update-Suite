# ROG Ally X Professional Update & Maintenance Suite

A comprehensive PowerShell script for maintaining and updating your ROG Ally X handheld gaming device.

## Features

- Automated Windows Updates
- Driver Updates
- BIOS Updates
- System Maintenance
- Performance Optimization
- Comprehensive Logging

## Usage

See [USAGE_GUIDE.md](USAGE_GUIDE.md) for detailed instructions.

## Requirements

- Windows OS
- PowerShell 5.1 or later
- Administrator privileges

## Installation

1. Download `Update-Ally-Pro.ps1`
2. Run PowerShell as Administrator
3. Execute the script
